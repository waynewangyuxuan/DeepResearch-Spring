# Research Dashboard — Module Meta

> Markdown-driven 极简研究仪表盘的设计与实现模式

## 文件清单

| 文件 | 内容 |
|------|------|
| KNOWLEDGE.md | 架构、页面设计、设计系统、数据流 |
| EVIDENCE.md | 设计决策来源（Evidence.dev、Tufte、MDX） |

- **context-os-bootstrap**：先建 Context OS，再建 Dashboard
